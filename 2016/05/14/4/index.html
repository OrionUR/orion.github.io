快捷键合集

2016-05-14   |   Tech   |  

这里纪录一些我使用各种软件以及操作系统上的快捷键，以此来简化操作
主要针对OS X，Windows和Linux不会涉及

OS X

  • 新增邮件Cmd + Sf + I
  • 选取文字翻译Cmd + Crl + D
  • 全屏截图Cmd + Sf + 3
  • 范围截图Cmd + Sf + 4
  • 隐藏窗口Cmd + H
  • 清倒废纸篓Cmd + Sf + Del
  • 删除光标后一个字符Fn + Del

Chrome

  • 新建标签页Cmd + T
  • 在后台打开链接Cmd + 目标
  • 打开链接并跳转Cmd + Sf + 目标
  • 重新打开关闭的标签页Cmd + Sf + T
  • 关闭标签页Cmd + W
  • 向左移动标签页Cmd + Op + Left
  • 向右移动标签页Cmd + Op + Right
  • 打开历史纪录Cmd + Y
  • 切换到地址栏Cmd + L
  • 刷新Cmd + R
  • 查找Cmd + F
  • 查找下一条Cmd + G
  • 查找上一条Sf + Enter

CLion

  • 对齐缩进（Align Indentation）：CMD + Op + L
